# Dormitory-Management-System--
这款基于微信小程序的宿舍管理系统，是校园宿舍管理的得力助手。学生可通过它在线申请入住、报修，提交晚归请假等信息。宿管人员能快速审核处理，查看学生住宿信息、卫生检查结果。还能发布宿舍通知，确保消息及时传达。系统设有违规记录功能，对宿舍安全隐患、违纪行为有效监管。它打破时间空间限制，简化管理流程，让宿舍管理更高效、便捷，营造安全有序的宿舍环境。
